
# Veo3 Static Site (VN + Settings + Actions)
- Chạy trực tiếp bằng `index.html` (click đúp là chạy).
- Nút ⚙️ Settings để nhập/đổi `GEMINI_API_KEY` (LocalStorage).
- Đã kèm workflow GitHub Actions để deploy Pages tự động.

## Deploy GitHub Pages (Actions)
1. Tạo repo mới (Public), push toàn bộ thư mục.
2. Vào **Settings → Pages** chọn **Source = GitHub Actions**.
3. Mỗi lần push `main`, site tự deploy.
